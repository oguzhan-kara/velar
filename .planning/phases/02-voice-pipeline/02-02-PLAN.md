**1. Create `app/voice/router.py`** with two authenticated endpoints:

```python
router = APIRouter(tags=["voice"])
```

**Endpoint 1: `POST /voice`**
- Parameters: `audio: UploadFile = File(...)`, `current_user: CurrentUser = Depends(get_current_user)`
- Pipeline: Read audio bytes -> STT -> Claude -> TTS -> return audio
- Steps:
  1. `audio_bytes = await audio.read()`
  2. `stt_result = await stt_service.transcribe(audio_bytes)` (import get_stt_service from stt.py and call it)
  3. If `not stt_result.text.strip()`: raise HTTPException(422, "No speech detected in audio")
  4. `response_text = await run_conversation(user_text=stt_result.text, history=[])`
  5. `audio_response = await tts_service.synthesize(text=response_text, language=stt_result.language)`
  6. Return `StreamingResponse(io.BytesIO(audio_response), media_type="audio/mpeg", headers={"X-Transcript": stt_result.text, "X-Response-Text": response_text, "X-Detected-Language": stt_result.language})`
- Error handling: STT failures -> 422, Claude failures -> 502/503 (from conversation.py), TTS failures -> 500 with message "TTS generation failed"

**Endpoint 2: `POST /chat`**
- Parameters: `request: ChatRequest` (body), `current_user: CurrentUser = Depends(get_current_user)`
- Pipeline: Text -> Claude -> TTS -> return JSON
- Steps:
  1. `response_text = await run_conversation(user_text=request.message, history=request.history)`
  2. `audio_response = await tts_service.synthesize(text=response_text, language=request.language or "tr")`
  3. `audio_base64 = base64.b64encode(audio_response).decode()`
  4. Return `ChatResponse(text=response_text, audio_base64=audio_base64, detected_language=request.language or "tr")`
